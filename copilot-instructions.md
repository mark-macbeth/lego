# LEGO Art Repository Instructions

This repository contains renders and models of Digital LEGO creations built using BrickLink Studio. It's primarily an art portfolio, not traditional code.

## Publishing Process

When assisting with publishing tasks:

- Focus on updating markdown files in `renders/`, `vignettes/`, and root documentation
- Help generate descriptions for new creations
- Assist with organizing image assets in `images/`
- Maintain model files in `models/` without unnecessary modifications
- Use semantic search to find related content across vignettes

## Content Guidelines

- Preserve the artistic intent of each vignette
- Ensure README files are descriptive and engaging
- Keep file structures organized by theme/project
- Avoid code-related assumptions; treat as documentation/art repo

## Automation Opportunities

- Suggest ways to batch update vignette descriptions
- Help create templates for new entries
- Assist with cross-referencing between related renders
